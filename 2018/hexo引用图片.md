在hexo3以上的版本, 可以直接在Markdown文件中插入图片

首先在博客总目录的_config.yml文件中，将**post_asset_folder**设置为**true**

这样在创建文件的时候会同时在source文件夹中生成一个同markdown名的文件夹

原本markdown插入图片的方法：

```
![图片描述\](图片位置\)
```

想要插入本地图片，需要将图片放在同名文件夹中，并将插入图片语句替换为：

```
{% asset_img 图片名.jpg 图片描述 %}
```

其中图片名处不需要写路径，方便又轻松
